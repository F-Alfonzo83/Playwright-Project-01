What is this?
Playwright Project aimed to test a basic E-Commerce layout site hosted on "Rahul Shetty Academy". 

OBJECTIVE:
Engineering goals include Page Object Model structure, Test reliability, separation of concerns,
single responsibility approach, "Do not repeat yourself" practices, "You Ain't Gonna Need it" approach and best practices.

BASE:
The project contains configuration modules, YAML config files, JSON credential files, pytest fixtures and more. 

STATUS:
Functional and Operative
